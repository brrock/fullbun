# React, react router, shadcn and hono example. 100% Bun, no extra bundler. 
Yep the title doesn't lie, no vite here. All served and bundled by bun. No server or client directory, everything is in one src/ dir.

## Running - dev
Make sure install deps `bun i`
```sh 
bun dev
```
## Running - prod
No build step 
```sh 
bun run start
```
